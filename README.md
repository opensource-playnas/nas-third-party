# nas-third-party
NAS项目中使用的第三方库，依据开源协议进行开源


# 编译
 - 本仓库代码基于 chromium的 103.0.5060.134 版本，编译环境详见chromium官方网站

# 目录结构说明
  build\args 编译相关参数
  common 本项目下的其他工程依赖公共文件
  cpp_redis redis厦工源码
  dlna_server 用于投屏服务，详细见目录下文档
  download_service 远程下载服务，详细见目录下文档
  media_server 媒体服务，获取媒体文件相关信息，详细见目录下文档
  third_party  编译需要的第三方文件，由于体积太大。有些依赖只标注依赖的版本，需要自行下载。
